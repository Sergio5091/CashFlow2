Conditions d'Utilisation de CashFlow

Bienvenue sur CashFlow ! Ces Conditions d'Utilisation ("Conditions") régissent votre accès et votre utilisation de la plateforme CashFlow (le "Service"), y compris le site web, les applications mobiles et tous les contenus, fonctionnalités et services offerts sur ou via le Service.

1. Acceptation des Conditions

En accédant ou en utilisant le Service, vous acceptez d'être lié par ces Conditions, ainsi que par notre Politique de Confidentialité. Si vous n'acceptez pas toutes ces Conditions, vous ne devez pas utiliser le Service.

---

2. Services Offerts

CashFlow est une plateforme qui offre l'accès à une bibliothèque de contenus vidéo (films et séries) et un programme de parrainage. L'accès complet à certains contenus et fonctionnalités est conditionné par un paiement unique.

---

3. Comptes Utilisateurs

  • Création de Compte : Pour accéder à certaines fonctionnalités du Service, vous devrez créer un compte en fournissant un nom complet, un numéro de téléphone et un mot de passe. Vous garantissez que toutes les informations fournies sont exactes, complètes et à jour.
  • Sécurité du Compte : Vous êtes seul responsable de la confidentialité de votre mot de passe et de toutes les activités qui se produisent sous votre compte. Vous devez nous informer immédiatement de toute utilisation non autorisée de votre compte.
  • Unicité du Compte : Il n'est pas possible de créer plusieurs comptes avec le même numéro de téléphone.
---

4. Paiements et Accès au Contenu

  • Activation de l'Accès : L'accès complet aux contenus vidéo est activé après un paiement unique de 2000 FCFA via MoMo USSD.
  • Procédure de Paiement : Vous devrez effectuer le paiement au numéro indiqué sur la plateforme. Après le paiement, vous devrez saisir l'ID de transaction sur votre tableau de bord utilisateur.
  • Validation du Paiement : Votre accès sera activé après vérification et validation de votre transaction par notre équipe administrative. Ce processus peut prendre jusqu'à 30 minutes. Tant que le paiement n'est pas validé par l'administrateur, l'accès aux contenus vidéo reste restreint.
  • Non-Remboursable : Les paiements effectués pour l'accès aux contenus sont non-remboursables.
---

5. Code de Parrainage

  • Génération de Code : Lors de votre inscription, un code de parrainage unique est généré pour votre compte.
  • Utilisation du Code : Vous pouvez partager ce code avec d'autres personnes pour qu'elles l'utilisent lors de leur inscription.
  • Vérification du Code : Si un utilisateur s'inscrit avec un code de parrainage invalide ou n'appartenant pas à un utilisateur existant (à l'exception du code "cashflow"), son parrain sera automatiquement défini sur "cashflow".
  • Gains de Parrainage : Les détails et conditions des gains liés au parrainage seront disponibles dans une section dédiée de votre tableau de bord utilisateur.
